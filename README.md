# My Data Selfie App

## 1. Overview
In this project, we explore some cool Javascripts functionalities, like API, Geolocation, POST requests, and so on.

## 2. Requirements

You need to have NodeJs and NPM installed.

## 3. Running the project:

```
npm install
npm run dev
```
